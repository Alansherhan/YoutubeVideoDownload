# youtube_video_download

### 🚀 Initial Release - v1.0.0

Welcome to the first release of the **YouTube Video & Audio Downloader**! 

#### ✨ Core Features
* **Link Parsing:** Instantly fetch video metadata by pasting a YouTube link.
* **Clipboard Integration:** "Paste from Clipboard" button for quick access.
* **Format Selection:** Choose from multiple video formats (muxed) and audio-only formats before downloading.
* **Real-time Tracking:** Clean downloading state with live percentage and a progress bar.
* **Library Management:** View, play, and slide-to-delete your downloaded media directly from the built-in Library tab.
* **Modern UI:** Sleek, dark-mode default interface with card-based designs and smooth navigation.

#### 📦 Installation (Android)
1. Download the `app-release.apk` file attached in the assets below.
2. Transfer it to or open it directly on your Android device.
3. If prompted by Android, allow installation from "Unknown Sources" for your browser/file manager.
4. Launch the app, grant the requested storage permissions on your first download, and enjoy!

*(Note: Don't forget to drag and drop the `app-release.apk` file from your `build/app/outputs/flutter-apk/` folder into the binaries section at the bottom of the release page!)*
